# 🌐 Web Development Assignment – GTG Perfumes

## 📌 Project Overview

This project involves translating a provided Figma design into a fully functional and responsive webpage using core web technologies. The primary focus is on accurately recreating the visual design while maintaining clean code practices and ensuring smooth user interaction.

---

## 🎯 Objective

The aim of this assignment is to demonstrate the ability to:

* Convert a UI design into a working webpage
* Maintain visual accuracy with respect to the original design
* Write structured, readable, and maintainable code
* Implement responsive layouts across multiple screen sizes

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS only)

> Note: No external frameworks or libraries (such as React or Vue) have been used.

---

## 📁 Project Structure

```
/project-folder
   /assets
   /css
   /js
   index.html
```

---

## 🎨 Design Implementation

* The layout closely follows the provided Figma design
* Spacing, typography, and alignment have been adjusted to closely match the reference
* Images used in the project are exported directly from the design file

---

## 📱 Responsiveness

The webpage adapts to different screen sizes:

* Desktop
* Tablet
* Mobile

Media queries and flexible layouts (Flexbox/Grid) are used to ensure proper scaling and alignment.

---

## ⚙️ Features & Functionality

### 🔹 Navigation

* Responsive header
* Hamburger menu for smaller screens

### 🔹 Product Section

* Interactive image gallery:

  * Arrow navigation
  * Dot indicators
  * Clickable thumbnails
* Dynamic product selection:

  * Multiple fragrance options
  * Different purchase types
* Add-to-cart link updates dynamically based on user selection

### 🔹 Subscription Options

* Expandable sections for different subscription plans
* UI updates based on selected option

### 🔹 Statistics Section

* Animated counters that increment when the section enters the viewport

### 🔹 Comparison Table

* Implemented using semantic HTML `<table>` structure
